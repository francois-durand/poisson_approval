DictPrintingInOrder¶

class poisson_approval.DictPrintingInOrder[source]¶

A dictionary that prints in the order of the keys.

Examples

>>> d = DictPrintingInOrder({'b': 'x', 'a': 'y', 'c': 'z'})
>>> print(d)
{a: y, b: x, c: z}
>>> print(repr(d))
{'a': 'y', 'b': 'x', 'c': 'z'}
