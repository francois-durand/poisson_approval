SetPrintingInOrder

class poisson_approval.SetPrintingInOrder[source]

A set that prints in order.

Examples

>>> s = SetPrintingInOrder({'b', 'a', 'c'})
>>> print(s)
{a, b, c}
>>> print(repr(s))
{'a', 'b', 'c'}
