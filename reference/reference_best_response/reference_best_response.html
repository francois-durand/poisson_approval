BestResponse

class poisson_approval.BestResponse(tau, ranking)[source]

Best response for a given ordinal type of voter (abstract class).

The main objective of this class is to compute utility_threshold. The subclasses implement the best response in a specific voting rule.

Parameters

  • tau (TauVector) – A tau-vector.

  • ranking (str) – Voter’s ranking, e.g. 'abc'.

i, j, k

The first (resp. second, third) candidate in ranking. E.g. a.

Type

str

ij, ik, jk

The ballots with two candidates. E.g. ab.

Type

str

tau_i, tau_j, tau_k, tau_ij, tau_ik, tau_jk

The values of the tau-vector.
