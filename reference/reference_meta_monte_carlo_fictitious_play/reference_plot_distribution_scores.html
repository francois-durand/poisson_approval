plot_distribution_scores

poisson_approval.plot_distribution_scores(results, voting_rule=None, **kwargs)[source]

Plot the distribution (CDF) of the scores for the winner, the challenger and the loser.

Parameters

  • results (dict) – Results of monte_carlo_fictitious_play(), with at least the setting MCS_DECREASING_SCORES.

  • voting_rule (str) – The voting rule (or None if this parameter was not given to monte_carlo_fictitious_play()).

  • kwargs – Other keyword arguments are passed to the function step of matplotlib.

Examples

>>> meta_results = monte_carlo_fictitious_play(
...     factory=RandProfileHistogramUniform(n_bins=1),
...     n_samples=1,
...     n_max_episodes=10,
...     monte_carlo_settings=[MCS_DECREASING_SCORES],
... )
>>> plot_distribution_scores(meta_results)
