RandProfileOrdinalUniformÔÉÅ

class poisson_approval.RandProfileOrdinalUniform(orders=None, d_order_fixed_share=None, **kwargs)[source]ÔÉÅ

A random factory of ordinal profiles (ProfileOrdinal) following the uniform distribution.

Parameters

  • orders (list, optional) ‚Äì These orders will have a variable share. They can be rankings, e.g. 'abc', or weak orders, e.g. 'a~b>c'. Default: all rankings.

  • d_order_fixed_share (dict, optional) ‚Äì A dictionary. For each entry order: fixed_share, this order will have at least this fixed share. The total must be lower or equal to 1.

  • kwargs ‚Äì These additional arguments will be passed directly to ProfileOrdinal.

Examples

Basic usage:

>>> initialize_random_seeds()
>>> rand_profile = RandProfileOrdinalUniform()
>>> profile = rand_profile()
>>> print(profile)
<abc: 0.4236547993389047, acb: 0.12122838365799216, bac: 0.0039303209304278885, bca: 0.05394987214431912, cab: 0.1124259903007756, cba: 0.2848106336275805> (Condorcet winner: a)

Using the optional parameters:

>>> rand_profile = RandProfileOrdinalUniform(
...     orders=['abc', 'acb'], d_order_fixed_share={'acb': 0.8, 'a~b>c': 0.1},
...     voting_rule=PLURALITY)
>>> profile = rand_profile()
>>> print(profile)
<abc: 0.0645894113066656, acb: 0.8354105886933344, a~b>c: 0.1> (Condorcet winner: a) (Plurality)

For more examples, cf. RandSimplexUniform.
